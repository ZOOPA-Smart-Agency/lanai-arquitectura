LANAI · herramientas de marca

Dos skills para crear con la marca.

Se instalan en Claude Code. Una genera contenido ya on-brand; la otra lo valida contra la arquitectura de marca. El resultado es siempre texto.

Requisito: Claude Code instalado. Las skills no cambian nada visual; entregan copy, guiones y conceptos.

/lanai-studio
Genera

Concibe y escribe cualquier pieza de LANAI ya on-brand. Inventa eventos y campañas, redacta emails y LP letters, construye el guion de una presentación. Aplica la voz, Smart Brevity y se autovalida antes de entregar.

Cómo instalarlas

  1. Descarga el .zip de la skill.
    Un archivo por skill. Puedes instalar las dos.
  2. Descomprímelo dentro de ~/.claude/skills/
    Para uso global. O en .claude/skills/ del proyecto si la quieres solo ahí. Debe quedar una carpeta lanai-studio/ con su SKILL.md dentro.
  3. Reinicia Claude Code.
  4. Invócala con /lanai-studio o /lanaizar.
    También se activan solas al redactar o validar contenido de LANAI.
